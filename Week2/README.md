This is a Bare Bone interpreter.

Commands:

  clear -> declare a variable
  
  incr -> increase a variable by 1
 
  decr -> decrease a variable by 1
  
  while -> while loop
 
Syntax:
  Every line has to end with ;
  clear:
    clear "variable name";
  incr:
    incr "variable name";
  decr:
    decr "variable name";
  while:
    while "variable name" not <intiger> do;
 
    .
 
    .
  
    .
 
    ends;
